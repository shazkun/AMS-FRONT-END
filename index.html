âš™ï¸ Setup & Install

Prerequisites

ToolVersion
Flutter3.x
Node.js18+
MySQL8.0+

1. Clone the repository

git clone https://github.com/your-username/ams.git
cd ams

2. Configure the backend

# Install dependencies
cd backend
npm install

# Create .env file
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=ams_db
JWT_SECRET=your_jwt_secret
PORT=3000

3. Run the database migration

mysql -u root -p ams_db < database/schema.sql

4. Start the backend server

npm start
# Server runs on http://localhost:3000

5. Configure the Flutter app

Open lib/config/api_config.dart and update the URL:

static const String _devUrl = 'http://your-backend-url.com';

6. Run the Flutter app

flutter pub get
flutter run

ğŸ‘¨â€ğŸ« Tutorial: Teacher App

Logging In

Use the username and password created for you by the admin.

Viewing Your Classes

The home screen lists all your assigned classes showing grade, section, and student count.

Opening an Attendance Session

  1. Tap a class
  2. Tap "Start Attendance"

    A session opens automatically for today's date. You can only have one session per class per day.

Scanning QR Codes

  1. Inside an open session, tap "Scan QR"
  2. Point camera at the student's QR code
  3. Confirmation appears

    Student is marked Present with time recorded.

âš ï¸
QR codes have a 30-second cooldown. Scanning the same student twice within 30 seconds will be rejected to prevent accidental duplicates.

SF2 Attendance Report

  1. Open a class
  2. Tap "SF2 Report"
  3. Select month and year

    The report shows a grid of all students vs. days in the month.

    Each session shows date, status (Present/Absent), and time marked.

Your QR Code

  1. Go to Profile
  2. Show your QR code to the teacher

    They will scan it to mark you present during an attendance session.
